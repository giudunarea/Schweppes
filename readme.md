# Config
## Install node modules
``` npm install ```
## Run server
``` nodemon server.js ```
## Build the css
``` npm run dev```

# Documentatie proiectului: Giu Projects

## Introducere: 
### Scopul acestui proiect este de a explora orașele din România și de a evidenția atracțiile lor, cum ar fi parcuri, muzee, obiective istorice și alte destinații turistice. Această documentație va servi ca ghid pentru călătorii care sunt interesați să viziteze diferite orașe din România și doresc să profite la maximum de călătoria lor.
### Tehnologii:
1.    HTML
2.    JavaScript
3.    ExpressJS
4.    CSS
5.    Tailwind CSS
6.    Flowbite

### Orase incluse in acest proiect:
1.    Bucuresti
2.    Brasov
3.    Iasi
4.    Galati
5.    Arad
6.    Constanta

### Atractii, pentru fiecare oras am adaugat:
1.    Parcuri
2.    Păduri
3.    Mall-uri
4.    Monumente
5.    Clădiri
